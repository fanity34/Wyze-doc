<AccordionGroup>

  <Accordion title=" O que é a WyzeHost?">
    A WyzeHost é uma empresa especializada em hospedagem de servidores, oferecendo soluções de alta performance para games, websites e aplicações diversas.
  </Accordion>

  <Accordion title="O que é uma vps e para que serve? ">
    O VPS é um servidor virtual que permite a hospedagem de aplicações sem a necessidade de você utilizar o seu computador. 
    É uma excelente solução para sites, banco de dados, bots, softwares e servidores de jogos como: Fivem, Rust, Valheim, Minecraft, DayZ, Counter-Strike, etc
  </Accordion>

  <Accordion title="Quais formas de pagamento são aceitas?">
    Aceitamos PIX, Cartão, PicPay, PayPal, Mercado Pago, Boleto e outras opções disponíveis no checkout.
  </Accordion>

  <Accordion title="Qual o preço de uma vps?">
    O preço de uma VPS depende dos recursos incluídos, como cpu, ram e armazenamento. Você pode conferir todos os nossos planos em [wyzehost.com.br/vps](https://www.wyzehost.com.br/vps), com opções a partir de R$80.
  </Accordion>

  <Accordion title="Onde encontro os dados de acesso da minha vps?">
    Os dados são entregue no seu email de forma imediata logo após a confirmação do pagamento.
  </Accordion>

    <Accordion title="Como conectar no vps?">
    O modo de conexão é diferente dependendo do sistema operacional instalado. Temos um guia prático mostrando o processo em [Windows](/vps/como-conectar-vps) e no [Linux](/vps/como-conectar-vps-linux).
  </Accordion>

  <Accordion title="Onde ficam localizados os servidores?">
    Nossos servidores estão estrategicamente localizados em Cotia-SP, proporcionando o melhor desempenho de conexão e a máxima estabilidade disponível no Brasil.
  </Accordion>

  <Accordion title="Vocês realizam reembolso?">
    Sim, estamos comprometidos com a satisfação dos nossos clientes. Caso você não esteja completamente satisfeito com nossos serviços, oferecemos um período de até 7 dias para solicitar o reembolso via ticket em nosso site.
  </Accordion>

    <Accordion title="Como alterar a senha do vps?">
    Tem 2 formas: diretamente pelo painel da vps ou dentro da própria vps. Para conferir o passo a passo, clique [AQUI](/painel/trocar-senha-vps).
  </Accordion>

  <Accordion title="Perco meus arquivos após realizar upgrade no painel do VPS?">
    Não, todos os arquivos são mantidos normalmente. Para aplicar as novas configurações, é necessário apenas reiniciar o VPS.
  </Accordion>

  <Accordion title="Existe um painel de controle disponível para o serviço? ">
    Sim, Todos os nossos VPS e Semi-Dedicado possuem painel incluso. Esse recurso permite gerenciar e controlar facilmente as configurações do seu servidor.
  </Accordion>
